HTML (HyperText Markup Language)

O HTML é uma linguagem de marcação utilizada na construção de páginas web. É a espinha dorsal de qualquer website, definindo a estrutura do conteúdo de forma hierárquica e semântica.

Os navegadores web recebem documentos HTML de um servidor ou do armazenamento local e os interpretam para exibir o conteúdo multimídia. O HTML, por si só, não é uma linguagem de programação; ele não pode criar funcionalidades dinâmicas. Para isso, utiliza-se o JavaScript. Para estilizar a aparência, usa-se o CSS.

História

O HTML foi criado por **Tim Berners-Lee** em 1991, com a primeira versão formalmente documentada em 1993. Sua evolução foi marcada por versões como HTML 2.0, HTML 3.2, HTML 4.01, e o mais recente, **HTML5**, lançado em 2014.

Principais Tags

Algumas das tags mais utilizadas e essenciais no desenvolvimento web são:

  • <p>: Representa um parágrafo.
  • <h1> a <h6>: Tags de cabeçalho para diferentes níveis de importância.
  • <a>: Cria links para outras páginas ou recursos.
  • <img>: Incorpora imagens na página.
  • <ul> e <li>: Utilizadas para criar listas não ordenadas.
